# 🌺 Cattleyas Bifoliadas Brasileñas

&gt; Periplo de cotejamiento entre ilustraciones de IA e imágenes reales

Una galería web elegante dedicada a las orquídeas **Cattleya** de hoja doble nativas de Brasil, comparando ilustraciones generadas por inteligencia artificial con fotografías botánicas reales.

![Estado](https://img.shields.io/badge/estado-activo-success)
![Tecnología](https://img.shields.io/badge/tecnología-HTML%2FCSS%2FJS-blue)
![Licencia](https://img.shields.io/badge/licencia-MIT-green)

## ✨ Características

- **🎨 Diseño vintage elegante** — Paleta de colores inspirada en herbarios clásicos (crema, pergamino, oro, tonos orquídea)
- **🔍 Sistema de filtros** — Navegación por especies: *C. amethystoglossa*, *C. bicolor*, *C. dormaniana*, *C. guttata*, *C. harrisoniae*, *C. intermedia*, *C. kerrii*, *C. loddigesii*, *C. schilleriana*, *C. tigrina*, *C. walkeriana*
- **💡 Lightbox interactivo** — Visualización ampliada de imágenes con navegación entre pares
- **📊 Comparación visual** — Layout lado a lado (ilustración IA | fotografía real)
- **📱 Responsive** — Adaptable a dispositivos móviles

## 🛠️ Tecnologías

- **HTML5** semántico
- **CSS3** con variables CSS y Grid/Flexbox
- **JavaScript vanilla** para interactividad (filtros, lightbox, navegación)
- **Fuentes:** Playfair Display, Cormorant Garamond (Google Fonts)
- **Datos embebidos** en base64 para imágenes
